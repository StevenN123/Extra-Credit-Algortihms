💰 Fractional Knapsack

Greedy Algorithm #1 - Maximizing Value with Fractions

📋 Problem Description

Given n items each with a weight and a value, and a knapsack with capacity W, find the maximum total value that can be placed in the knapsack. Unlike the 0/1 knapsack problem, you can take fractions of items.

Formal Definition:

  • Given: Items i = 1...n with weight wᵢ and value vᵢ
  • Given: Knapsack capacity W
  • Find: Maximum total value such that total weight ≤ W
  • Constraint: Can take any fraction xᵢ of item i where 0 ≤ xᵢ ≤ 1
Maximize: Σ vᵢ·xᵢ subject to Σ wᵢ·xᵢ ≤ W, 0 ≤ xᵢ ≤ 1
O(n log n)
Time Complexity
O(1)
Space Complexity
Optimal
Greedy Choice

Example:

Item Weight Value Ratio (v/w)
1 10 60 6.0
2 20 100 5.0
3 30 120 4.0

Capacity: 50

Optimal solution: Take items 1 and 2 completely, and 2/3 of item 3 = 60 + 100 + 80 = 240

🐍 Python Implementation

def fractional_knapsack(items, capacity):
    """
    Solve fractional knapsack problem using greedy approach
    
    Args:
        items: List of tuples (weight, value)
        capacity: Maximum weight capacity
    
    Returns:
        tuple: (maximum_value, items_taken_details)
    """
    # Step 1: Calculate value/weight ratio for each item
    # Create a new list with (weight, value, ratio) for sorting
    items_with_ratio = []
    for weight, value in items:  # Loop through each item
        ratio = value / weight    # Calculate value per unit weight
        items_with_ratio.append((weight, value, ratio))  # Add to list
    
    # Step 2: Sort items by ratio in descending order (highest ratio first)
    # key=lambda x: x[2] means sort by the third element (ratio)
    # reverse=True means descending order
    items_with_ratio.sort(key=lambda x: x[2], reverse=True)
    
    total_value = 0  # Initialize total value accumulator
    remaining = capacity  # Track remaining capacity
    items_taken = []  # List to store what we took
    
    # Step 3: Take items greedily based on ratio
    for weight, value, ratio in items_with_ratio:
        if remaining >= weight:
            # Case 1: Item fits completely - take all of it
            total_value += value  # Add full value
            remaining -= weight  # Reduce remaining capacity
            items_taken.append(f"Took whole item (weight:{weight}, value:{value})")
        else:
            # Case 2: Item doesn't fit completely - take fraction
            fraction = remaining / weight  # Calculate fraction that fits
            total_value += value * fraction  # Add fractional value
            items_taken.append(f"Took {fraction:.2f} of item (value:{value * fraction:.2f})")
            break  # Knapsack is full, exit loop
    
    return total_value, items_taken  # Return results
