Fractional Knapsack

Greedy Algorithm #1 - Maximizing Value with Fractions

Problem Description

Given n items each with a weight and a value, and a knapsack with capacity W, find the maximum total value that can be placed in the knapsack. Unlike the 0/1 knapsack problem, you can take fractions of items.

Formal Definition:

  • Given: Items i = 1...n with weight wᵢ and value vᵢ
  • Given: Knapsack capacity W
  • Find: Maximum total value such that total weight ≤ W
  • Constraint: Can take any fraction xᵢ of item i where 0 ≤ xᵢ ≤ 1
Maximize: Σ vᵢ·xᵢ subject to Σ wᵢ·xᵢ ≤ W, 0 ≤ xᵢ ≤ 1
O(n log n)
Time Complexity
O(1)
Space Complexity

Example:

Item Weight Value Ratio (v/w)
1 10 60 6.0
2 20 100 5.0
3 30 120 4.0

Capacity: 50

Optimal solution: Take items 1 and 2 completely, and 2/3 of item 3 = 60 + 100 + 80 = 240
