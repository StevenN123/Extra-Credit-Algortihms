🎒 0/1 Knapsack

Dynamic Programming #2 - Maximizing Value with Indivisible Items

📋 Problem Description

Given n items, each with a weight and a value, and a knapsack with capacity W, find the maximum total value that can be placed in the knapsack. Items are indivisible - you either take an item (1) or leave it (0). This is called the 0/1 knapsack problem.

Formal Definition:

  • Given: Items i = 1...n with weight wᵢ and value vᵢ
  • Given: Knapsack capacity W
  • Find: Maximum Σ vᵢ·xᵢ subject to Σ wᵢ·xᵢ ≤ W
  • Constraint: xᵢ ∈ {0,1} (either take whole item or leave it)
Maximize: Σ vᵢ·xᵢ subject to Σ wᵢ·xᵢ ≤ W, xᵢ ∈ {0,1}
O(n × W)
Time Complexity
O(n × W)
Space Complexity
O(W)
Space Optimized

Example:

Item 1
Weight: 2
Value: 12
Item 2
Weight: 1
Value: 10
Item 3
Weight: 3
Value: 20
Item 4
Weight: 2
Value: 15
Capacity: W = 5
Optimal solution: Take items 1, 2, and 4 (weights: 2+1+2=5, value: 12+10+15=37)

🐍 Python Implementation

Approach 1: 2D DP Table

def knapsack_01(items, capacity):
    """
    Solve 0/1 knapsack problem using 2D DP table
    
    Args:
        items: List of tuples (weight, value)
        capacity: Maximum weight capacity
    
    Returns:
        tuple: (maximum_value, list_of_items_taken)
    
    Time Complexity: O(n × W)
    Space Complexity: O(n × W)
    """
    n = len(items)  # Number of items
    
    # Create DP table with (n+1) rows and (capacity+1) columns
    # dp[i][w] = maximum value using first i items with capacity w
    # Initialize all cells to 0
    dp = [[0] * (capacity + 1) for _ in range(n + 1)]
    
    # Build table bottom-up
    for i in range(1, n + 1):  # For each item (1 to n)
        weight, value = items[i - 1]  # Get current item (i-1 because items is 0-indexed)
        
        for w in range(capacity + 1):  # For each possible capacity (0 to W)
            if weight <= w:
                # Current item can fit in the knapsack
                # Two choices:
                # 1. Don't take item i: value = dp[i-1][w]
                # 2. Take item i: value = value + dp[i-1][w-weight]
                dp[i][w] = max(
                    dp[i - 1][w],  # Don't take item
                    dp[i - 1][w - weight] + value  # Take item
                )
            else:
                # Current item too heavy - can't take it
                dp[i][w] = dp[i - 1][w]
    
    # Backtrack to find which items were taken
    items_taken = []  # List to store indices of taken items
    w = capacity  # Start from full capacity
    
    for i in range(n, 0, -1):  # Go backwards through items
        if dp[i][w] != dp[i - 1][w]:
            # If value changed, item i-1 was taken
            items_taken.append(i - 1)  # Add item index (0-based)
            w -= items[i - 1][0]  # Reduce remaining capacity
    
    items_taken.reverse()  # Reverse to get items in original order
    
    return dp[n][capacity], items_taken

Approach 2: Space-Optimized 1D DP

def knapsack_01_space_optimized(items, capacity):
    """
    Space-optimized version using 1D DP array
    
    Since each row only depends on the previous row,
    we can use a 1D array and update from right to left.
    
    Args:
        items: List of tuples (weight, value)
        capacity: Maximum weight capacity
    
    Returns:
        int: Maximum value achievable
    
    Time Complexity: O(n × W)
    Space Complexity: O(W)
    """
    n = len(items)  # Number of items
    
    # Create 1D DP array of size capacity+1
    # dp[w] = maximum value achievable with capacity w
    dp = [0] * (capacity + 1)
    
    # Process items one by one
    for i in range(n):  # For each item
        weight, value = items[i]  # Get current item
        
        # Traverse capacity backwards to avoid reusing the same item
        # If we go forwards, we might use the same item multiple times
        for w in range(capacity, weight - 1, -1):
            # Update dp[w] to max of:
            # 1. Not taking item: dp[w] (current value)
            # 2. Taking item: dp[w-weight] + value
            dp[w] = max(dp[w], dp[w - weight] + value)
    
    return dp[capacity]  # Return max value for full capacity

# This works because when we update from right to left,
# dp[w-weight] still represents the value WITHOUT the current item

Complete Example

# Example usage
if __name__ == "__main__":
    # Define items as (weight, value) pairs
    items = [(2, 12), (1, 10), (3, 20), (2, 15)]
    capacity = 5
    
    print("0/1 Knapsack Problem")
    print("=" * 50)
    print("Items (weight, value):")
    for i, (w, v) in enumerate(items):
        print(f"  Item {i+1}: weight={w}, value={v}")
    print(f"Knapsack Capacity: {capacity}")
    print()
    
    # Solve using 2D DP
    max_value, taken = knapsack_01(items, capacity)
    print(f"2D DP Solution:")
    print(f"  Maximum Value: {max_value}")
    print(f"  Items taken: {[i+1 for i in taken]} (indices {taken})")
    print(f"  Total weight: {sum(items[i][0] for i in taken)}")
    
    # Solve using space-optimized DP
    max_value_opt = knapsack_01_space_optimized(items, capacity)
    print(f"\nSpace-Optimized Solution:")
    print(f"  Maximum Value: {max_value_opt}")

📐 Recurrence Relation

dp[i][w] = max(dp[i-1][w], dp[i-1][w-wᵢ] + vᵢ) if wᵢ ≤ w
dp[i][w] = dp[i-1][w] if wᵢ > w

Where:

  • dp[i][w] = maximum value using first i items with capacity w
  • wᵢ = weight of item i
  • vᵢ = value of item i

Base Cases:

  • dp[0][w] = 0 for all w (no items → zero value)
