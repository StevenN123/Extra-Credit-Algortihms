Fibonacci Numbers

Dynamic Programming #1 - Multiple DP Approaches

Problem Description

The Fibonacci sequence is defined as: F(0) = 0, F(1) = 1, and F(n) = F(n-1) + F(n-2) for n > 1. Calculate the nth Fibonacci number efficiently using dynamic programming.

F(n) = F(n-1) + F(n-2), with F(0) = 0, F(1) = 1
O(2ⁿ)
Naive Recursive
O(n)
DP Approaches
O(1)
Space Optimized

Example:

Input: n = 10
Output: 55

Fibonacci Sequence up to n=10:

F(0)=0 F(1)=1 F(2)=1 F(3)=2 F(4)=3 F(5)=5 F(6)=8 F(7)=13 F(8)=21 F(9)=34 F(10)=55
