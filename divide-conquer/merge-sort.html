Merge Sort

Divide & Conquer Algorithm #1 - Stable O(n log n) Sorting

Problem Description

Sort an array using the divide and conquer approach: recursively divide the array into halves, sort them, and merge the sorted halves. Merge sort is a stable, comparison-based sorting algorithm with guaranteed O(n log n) performance.

How It Works:

  • Divide: Split the array into two halves at the middle index
  • Conquer: Recursively sort each half
  • Combine: Merge the two sorted halves into one sorted array
O(n log n)
Time Complexity
O(n)
Space Complexity
Stable
Sorting Property

Example:

Input: [64, 34, 25, 12, 22, 11, 90]
Output: [11, 12, 22, 25, 34, 64, 90]
