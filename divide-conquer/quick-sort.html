Quick Sort

Divide & Conquer Algorithm #2 - In-Place Sorting with Pivot

Problem Description

Sort an array using the divide and conquer approach: pick a pivot element, partition the array around the pivot (smaller elements on left, larger on right), and recursively sort the subarrays. Quick sort is an in-place sorting algorithm with excellent average-case performance.

How It Works:

  • Choose Pivot: Select a pivot element (random selection helps avoid worst-case)
  • Partition: Rearrange array so elements smaller than pivot go left, larger go right
  • Recursively Sort: Apply quick sort to left and right subarrays
O(n log n) avg
Time Complexity (Average)
O(n²)
Time Complexity (Worst)
O(log n)
Space Complexity
Note: Quick sort has O(n²) worst case (when pivot is always smallest or largest), but with random pivot selection, it's O(n log n) in practice and is often faster than merge sort due to better cache locality.

Example:

Input: [64, 34, 25, 12, 22, 11, 90]
Output: [11, 12, 22, 25, 34, 64, 90]
