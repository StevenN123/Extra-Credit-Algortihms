🧬 Genetic Algorithm (TSP)

NP Heuristic - Approximating the Traveling Salesman Problem

Problem Description

The Traveling Salesman Problem (TSP): Find the shortest possible route that visits each city exactly once and returns to the starting city. This is NP-hard, so we use a genetic algorithm heuristic.

O(g·n²)
Time Complexity*
O(p·n)
Space Complexity
~95%
Avg. Optimality

*where g = generations, p = population size, n = number of cities

Evolutionary Process

1️⃣ Initialization
2️⃣ Selection
3️⃣ Crossover
4️⃣ Mutation
5️⃣ Evolution

Algorithm Steps:

  1. Generate random population of routes
  2. Evaluate fitness (route distance)
  3. Select parents using tournament selection
  4. Apply ordered crossover (OX) to create offspring
  5. Mutate with swap mutation
  6. Replace population with new generation
  7. Repeat until convergence or max generations
